# Naturaleza Sagrada S.A.S. BIC - Website

Static corporate website for Naturaleza Sagrada S.A.S. BIC.

## Overview

This project is a full migration from a Laravel-based system to a static architecture optimized for performance, SEO, and deployment on Cloudflare Pages.

The website represents a handcrafted natural products brand inspired by traditional knowledge and cultural practices.

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Custom CSS

No frameworks or backend dependencies.

## Structure

- index.html → Home page
- productos.html → Product catalog
- quienes-somos.html → Company information
- contacto.html → Contact hub
- product pages → Individual product landing pages

## Assets

All static assets are located in:

assets/
- css/
- js/
- images/

## Deployment

Designed for Cloudflare Pages:

- No build step required
- Root deployment
- Static hosting only
